1. Prerequisites required.
	Install JRE 1.7 or above. To download JRE 1.7 http://www.oracle.com
	File write permission for a folder, where agent will be configured.
	MYSQL 5.0/5.5 should be installed.
2. Create Appedo agent's home folder
	D:\appedo_mysql_agent
3. Download the agent under D:\appedo_mysql_agent

4. Use Winrar/Winzip/Other tools to extract the downloaded setup

	tar -xzmvf appedo_mysql_agent_#VERSION#.tar.gz
	
5. Add Mysql Configurations:
	open mysql_config.properties then specify the LOCAL_LISTERN_PORT,USER_NAME and  USER_PASSWORD as per monitor machine
	Example:
	LOCAL_LISTERN_PORT=3306
	USER_NAME=root
	USER_PASSWORD=sa
	
6. Slow Query configuration
    Add / Change the following in my.cnf and restart the mysqld
     --log-output=TABLE
     --slow-query-log=ON
     --long_query_time=10 (OPTIONAL. This is the default time in 10sec)
     Then restart mysqld. To avoid restart, use the below query
     set global log_output="TABLE";
	 set global slow_query_log=ON;
	 set global long_query_time=10; (OPTIONAL)
	 NOTE : Given mysqk username should have permission to access mysql.slow_log table.
	
7. Install SSL
	copy "%JAVA_HOME%\jre\lib\security\cacerts" cacerts_appedo_agent
	keytool -import -alias apm.appedo.com -keystore cacerts_appedo_agent -storepass changeit -file apm.appedo.com.crt -noprompt

	Note: "changeit" is the default store password for the "cacerts". If it changed, then use the new password in the above and below commands.

8. To run the agent:
	cd /D D:\appedo_mysql_agent\
	For Windows XP :
	java -Djavax.net.ssl.trustStore=cacerts_appedo_agent -Djavax.net.ssl.trustStorePassword=changeit -jar appedo_mysql_agent_#VERSION#.jar > logs/appedo_mysql_agent_#VERSION#_%date:~10,4%%date:~7,2%%date:~4,2%_%time:~0,2%%time:~3,2%.log
	For Windows 7:
	java -Djavax.net.ssl.trustStore=cacerts_appedo_agent -Djavax.net.ssl.trustStorePassword=changeit -jar appedo_mysql_agent_#VERSION#.jar > logs/appedo_mysql_agent_#VERSION#_%date:~6,4%%date:~3,2%%date:~0,2%_%time:~0,2%%time:~3,2%.log
